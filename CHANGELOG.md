# Faraday::HotMock Change Log

## v0.6.0

- Added VCR mode

## v0.5.0

- Added scenarios
- Added an undocumented (for now) interpolation feature
- Adjusted headers included in mocked responses

## v0.4.0

- Added more convenience methods
- Added methods to check if mocking is enabled or disabled.
- Updated README with new examples, warning, and comparison with VCR
- Moved methods from adapter to main module for easier access & wider usage

## v0.3.0

- Added convenience methods to enable or disable mocking programmatically
- Updated documentation to include examples of the new convenience methods

## v0.2.0

- Bug and typo fixes

## v0.1.0

- Initial release of Faraday::HotMock
